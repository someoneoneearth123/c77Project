Visual Studio Code is a code editor which became very popular after release. Go to Visual Studio Code's website to install it. You can add extensions, themes and customize it however you want. To get started click open a folder, then open a folder create a file by pressing the button with a single file and plus button name it whatever you want and then start coding!



The console is a part of a browser where developers can log to. To open it, right click, select inspect, and then click console. You can run code in the console from the browser, and you can log things to it by using console.log() in javascript. Open the console to see a few messages right now.
